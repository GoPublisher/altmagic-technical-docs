# Alt Magic API

Generate descriptive, SEO-optimized alt text for images using our AI-powered service. The Alt Magic API automatically falls back from Gemini to OpenAI for maximum reliability, ensuring your images always get the alt text they need.

## üöÄ Quick Start

### 1. Get Your API Key

1. Sign up at [app.altmagic.pro](https://app.altmagic.pro)
2. Navigate to **API Keys** section
3. Generate your first API key

[View detailed step-by-step guide ‚Üí](/api-reference/api-key-generation)

### 2. Make Your First Request

```bash
curl -X POST "https://api.altmagic.pro/api-alt-text" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "image_url": "https://example.com/image.jpg",
    "user_id": "your-email@example.com"
  }'
```

### 3. Get Your Alt Text

```json
{
  "altText": "A close-up photo of a red apple on a wooden table",
  "CreditsAvailable": 49,
  "plan_type": "pro"
}
```

## üìö Examples

### Basic Usage

Generate alt text for a simple image:

```javascript
const response = await fetch('https://api.altmagic.pro/api-alt-text', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': API_KEY
  },
  body: JSON.stringify({
    image_url: 'https://example.com/product.jpg',
    user_id: 'user@example.com'
  })
});
```

const result = await response.json();
console.log(result.altText);
// Output: "A modern smartphone displayed on a white background"
```

### With Context

Provide additional context for better alt text:

```javascript
const response = await fetch('https://api.altmagic.pro/api-alt-text', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': API_KEY
  },
  body: JSON.stringify({
    image_url: 'https://example.com/recipe.jpg',
    blog_title: 'Homemade Pizza Recipe',
    seo_keywords: 'pizza, cooking, recipe, homemade',
    language_code: 'en',
    user_id: 'user@example.com'
  })
});
```

### Batch Processing

Process multiple images efficiently:

```javascript
const imageUrls = [
  'https://example.com/image1.jpg',
  'https://example.com/image2.jpg',
  'https://example.com/image3.jpg'
];

const results = await Promise.all(
  imageUrls.map(url => 
    api.generateAltText({
      image_url: url,
      user_id: 'user@example.com'
    })
  )
);

results.forEach((result, index) => {
  console.log(`Image ${index + 1}: ${result.altText}`);
});
```

## üîç Best Practices

### Image URLs
- Ensure images are publicly accessible
- Use HTTPS URLs when possible
- Avoid authentication-protected images

### Rate Limiting
- Implement exponential backoff
- Queue requests for high-volume usage
- Monitor your usage patterns

- Implement retry logic for transient errors
- Log errors for debugging

### Security
- Store API keys securely
- Use environment variables
- Rotate keys regularly
