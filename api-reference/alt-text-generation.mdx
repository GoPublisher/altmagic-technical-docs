Generate descriptive alt text for images using our AI-powered service. This endpoint automatically falls back from Gemini API to OpenAI if needed, ensuring reliable alt text generation for your images.

## Endpoint

```http
POST /api-alt-text
```

## Overview

The Alt Text Generation API creates descriptive, SEO-optimized alt text for images. It automatically deducts credits from your account and provides intelligent fallback between AI models for maximum reliability.

## Authentication

This endpoint requires authentication via API key:

- **API Key**: Include `x-api-key` header

### Header Parameter

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `x-api-key` | string | Yes | Your API key |

## Request Body

### Required Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `image_url` | string | Publicly accessible URL of the image to generate alt text for |
| `user_id` | string | Unique identifier (email) for the user requesting the alt text |

### Optional Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `blog_title` | string | Blog or article title to provide additional context |
| `seo_keywords` | string | Comma-separated keywords for SEO optimization |
| `language_code` | string | Language code (e.g., "en", "fr", "es") for generated alt text ([View all supported languages →](/api-reference/language-support)). If not provided, defaults to English (`en`) |

### Example Request

```json
{
  "image_url": "https://cdn.shopify.com/b/shopify-brochure2-assets/8e3de162e96c8505fd494f2d0046e82a.jpg",
  "blog_title": "testing_1",
  "seo_keywords": "For_company",
  "language_code": "en",
  "user_id": "abcxyz@gmail.com"
}
```

## Language Support

The Alt Magic API supports **200+ languages** for generating alt text. You can specify the desired language using the `language_code` parameter in your requests.

**Popular Languages:**
- **English** (`en`) - Default language (fallback when no language specified)
- **Spanish** (`es`) - Español
- **French** (`fr`) - Français
- **German** (`de`) - Deutsch
- **Chinese** (`zh-CN`, `zh-TW`) - 中文
- **Japanese** (`ja`) - 日本語
- **Korean** (`ko`) - 한국어
- **Arabic** (`ar`) - العربية
- **Hindi** (`hi`) - हिन्दी
- **Portuguese** (`pt`, `pt-BR`) - Português

[View complete language list →](/api-reference/language-support)

## Rate Limits

- **Limit**: 100 requests per user per minute
- **Identification**: By `user_id` if available, otherwise by IP address
- **Exceeded Response**: HTTP 429 with message "Too many requests from this User, please try again later."

## Responses

### 200 OK - Successful Generation

```json
{
  "altText": "A close-up photo of a red apple on a wooden table",
  "CreditsAvailable": 49,
  "plan_type": "pro"
}
```

**Response Fields:**
- `altText` - The generated alt text
- `CreditsAvailable` - Remaining credits for the user after this request
- `plan_type` - The user's subscription plan

### 403 Forbidden - No Credits Remaining

```json
{
  "message": "No credits remaining."
}
```

### 404 Not Found - User Does Not Exist

```json
{
  "message": "User not found."
}
```

### 429 Too Many Requests - Rate Limit Exceeded

```json
{
  "message": "Too many requests from this User, please try again later."
}
```

### 500 Internal Server Error

```json
{
  "success": false,
  "message": "Internal server error",
  "error": "Detailed error message here"
}
```

## Workflow

1. **Validate user exists** - Check if the provided `user_id` is valid
2. **Check credits** - Verify user has sufficient credits for the request
3. **Generate alt text** - Use Gemini API with automatic fallback to OpenAI
4. **Deduct credit** - Remove one credit from user's account
5. **Return results** - Provide generated alt text and updated credit count

## Code Examples

### cURL

```bash
curl -X POST "https://api.altmagic.pro/api-alt-text" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "image_url": "https://example.com/image.jpg",
    "user_id": "user@example.com",
    "blog_title": "My Blog Post",
    "seo_keywords": "technology, AI, automation",
    "language_code": "en"
  }'
```

### JavaScript

```javascript
const response = await fetch('https://api.altmagic.pro/api-alt-text', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    image_url: 'https://example.com/image.jpg',
    user_id: 'user@example.com',
    blog_title: 'My Blog Post',
    seo_keywords: 'technology, AI, automation',
    language_code: 'en'
  })
});

const result = await response.json();
console.log(result.altText);
```

### Python

```python
import requests

url = "https://api.altmagic.pro/api-alt-text"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
data = {
    "image_url": "https://example.com/image.jpg",
    "user_id": "user@example.com",
    "blog_title": "My Blog Post",
    "seo_keywords": "technology, AI, automation",
    "language_code": "en"
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result["altText"])
```

## Best Practices

- **Image URLs**: Ensure images are publicly accessible and not behind authentication
- **Rate Limiting**: Implement exponential backoff for retry logic
- **Error Handling**: Always check response status codes and handle errors gracefully
- **Credit Management**: Monitor your credit balance to avoid service interruptions
