# Documentation

Welcome to Alt Magic documentation! Learn how to use our AI-powered service to generate descriptive, SEO-optimized alt text for your images.

## 🚀 Getting Started

### What is Alt Magic?

Alt Magic is an AI-powered service that automatically generates descriptive alt text for images. Our service uses advanced AI models with automatic fallback to ensure reliable alt text generation for accessibility and SEO.

### Key Benefits

- **🤖 AI-Powered**: Uses Gemini API with automatic OpenAI fallback
- **💳 Credit-Based**: Pay only for what you use
- **⚡ Fast**: Generate alt text in seconds
- **🌍 Multi-Language**: Support for 200+ languages
- **🔒 Secure**: API key authentication with rate limiting

## 🌍 Language Support

The Alt Magic API supports **200+ languages** for generating alt text in your preferred language.

**Popular Languages:**
- **English** (`en`) - Default language (fallback when no language specified)
- **Spanish** (`es`) - Español
- **French** (`fr`) - Français
- **German** (`de`) - Deutsch
- **Chinese** (`zh-CN`, `zh-TW`) - 中文
- **Japanese** (`ja`) - 日本語
- **Korean** (`ko`) - 한국어
- **Arabic** (`ar`) - العربية
